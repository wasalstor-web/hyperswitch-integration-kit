إعداد المطوّر (محلي أو Supabase)

محلي + Prisma: من جذر المشروع npm run server:dev، وفي .env ضع DATABASE_URL وPUBLIC_APP_VERIFY_URL=http://localhost:5173، وفي web/.env.local ضع VITE_API_BASE_URL=http://localhost:8788.

Supabase: VITE_SUPABASE_URL وVITE_SUPABASE_ANON_KEY، وأسرار الدوال PUBLIC_APP_VERIFY_URL.
